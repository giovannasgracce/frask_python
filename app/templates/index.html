Bem-vindo ao Sistema Crod

Este é um sistema de gestão de dados simples e eficiente, onde você pode cadastrar, atualizar, conferir e deletar itens de forma prática. Use os botões acima para começar a gerenciar seus dados!



Como usar o Crod


    Para Cadastrar: Clique em "Cadastrar" no menu e preencha o formulário com as informações do item.

    Para Atualizar: Vá até "Atualizar", escolha o item que deseja modificar e edite as informações necessárias.

    Para Conferir: Use "Conferir" para visualizar todos os itens cadastrados no sistema.

    Para Deletar: Vá até "Deletar", selecione o item e confirme a exclusão.
